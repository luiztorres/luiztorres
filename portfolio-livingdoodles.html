  1. Início
  2. Living Doodles

Descrição do projeto

Infinite runner produzido durante o Global Game Jam (2015) no qual o personagem principal é um desenho que ganha vida e começa a correr através das páginas de um caderno. Seu objetivo é sobreviver, mas para isso deverá evitar os monstros e obstáculos. Para facilitar essa tarefa, o personagem conta com ajuda de um desenhista (player) que pode desenhar objetos para o auxiliar nessa jornada.

Características

  • Unity/C#
  • PC
